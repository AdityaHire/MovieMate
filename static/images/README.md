# Movie Poster Images

Place your movie poster images in this directory with the following names:
- poster1.jpg (or .png)
- poster2.jpg (or .png)
- poster3.jpg (or .png)

Recommended specifications:
- Image ratio: 2:3 (portrait)
- Recommended size: 400x600 pixels or higher
- Format: JPG or PNG
- File size: Keep under 500KB for faster loading

If images are not found, the system will automatically display placeholder icons.

## Where to find movie posters:
1. Free movie poster sites (ensure they're copyright-free)
2. Use your own movie collection posters
3. Create custom designs
4. Screenshot from movie databases (for personal/educational use)

## Example sources:
- Unsplash.com (search for "movie poster")
- Pexels.com (search for "cinema")
- Your own creative designs
